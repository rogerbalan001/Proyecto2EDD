Proyecto 2: Simulador de Cola de Impresión

📖 Descripción
Este sistema simula la gestión de documentos en una cola de impresión de un Sistema Operativo. Utiliza un Montículo Binario (Min-Heap) para organizar los documentos por prioridad (etiqueta de tiempo) y una Tabla de Dispersión (Hash Table) para permitir la cancelación rápida de documentos.

El sistema garantiza que los usuarios con mayor jerarquía y los documentos marcados como "prioritarios" sean atendidos primero por la impresora.
